THƯ MỤC GHI NHỚ RIÊNG CỦA USER

Mục đích:
- bạn có thể tạo file ghi nhớ, note, checklist, thông tin tạm hoặc tài liệu muốn giữ lại trong thư mục này

Luật bảo vệ mặc định:
- mọi agent đều bị cấm đọc / ghi / sửa / xoá / move / rename nội dung trong thư mục này
- chỉ được đụng vào khi user yêu cầu rất rõ hành động cụ thể và chỉ rõ file hoặc đường dẫn

Khuyến nghị:
- nếu bạn có file muốn giữ chắc chắn, hãy đặt vào thư mục `memo files/`
