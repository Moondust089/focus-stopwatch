# MỤC LỤC SỔ LỆNH

## File hướng dẫn chính
- `HUONG_DAN_DAY_DU_DUNG_COMMANDS_VA_AGENT.md`

## Sổ lệnh
- `agents/commands/sonnet_commands.md`
- `agents/commands/gemini_commands.md`
- `agents/commands/opus_commands.md`

## Dùng nhanh
- Muốn Sonnet tạo mission: dùng lệnh 1, 2, 3, hoặc 4 trong `sonnet_commands.md`
- Muốn Gemini thực thi mission thường: dùng lệnh 1 trong `gemini_commands.md`
- Muốn Gemini retry: dùng lệnh 2 trong `gemini_commands.md`
- Muốn Gemini chạy mission UI/STITCH: dùng lệnh 3 trong `gemini_commands.md`
- Muốn Opus phân tích escalation, chọn hướng sửa và cập nhật mission luôn: dùng lệnh 1 trong `opus_commands.md`
- Muốn chốt done: dùng lệnh 9 trong `sonnet_commands.md`
- Muốn kéo agent về đúng vai trò: dùng lệnh re-anchor tương ứng trong từng sổ lệnh hoặc file trong `agents/refresh/`
